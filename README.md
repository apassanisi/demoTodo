### Usage

| Use                    | Command              | URL                                            |
| ---------------------- | -------------------- | ---------------------------------------------- |
| Install locally        | `pnpm i && pnpm dev` |                                                |
| Start local dev server | `pnpm dev`           | [http://localhost:5173](http://localhost:5173) |
| Static build           | `pnpm build`         |                                                |
| Preview static build   | `pnpm serve`         | [http://localhost:4173](http://localhost:4173) |
